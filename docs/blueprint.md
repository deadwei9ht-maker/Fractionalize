# **App Name**: NFT Fractionalizer

## Core Features:

- NFT Contract Input: Allow users to input the NFT contract address they want to fractionalize.
- Token ID Input: Allow users to input the Token ID of the NFT they want to fractionalize.
- NFT Fractionalization: Fractionalize the specified NFT into 10,000 tradable tokens upon user request. Simulate the interaction with a smart contract for fractionalization.
- Shareable Link: Generate a shareable link that includes the Token ID of the fractionalized NFT. Allow users to copy this link to share with others.
- Uniswap Integration (Testnet): Provide a button that, when clicked, simulates the process of adding the fractionalized tokens to Uniswap on a testnet. Display an alert indicating the successful (simulated) creation of a Uniswap pool.

## Style Guidelines:

- Primary color: Vibrant magenta (#FF00FF), inspired by the energy and forward thinking inherent in NFTs.
- Background color: Very dark, desaturated purple (#0A0A1A) to make the other colors pop.
- Accent color: Bright cyan (#00FFEA) to complement the magenta.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean aesthetic.
- Use simple, line-based icons to represent NFT fractionalization and sharing.
- Center the content on the page with a card-based layout for focus.
- Subtle pulsing animation on the 'Fractionalize Now' button to draw attention.
